Контейнер веб-сервера запущен OK

Эта страница отдаётся контейнером nginx, собранным из собственного Dockerfile. Конфигурация сервера и эта страница скопированы внутрь образа на этапе сборки.

Контейнер с PostgreSQL запускается раньше веб-сервера и хранит файлы базы данных на хост-машине, поэтому данные не теряются при пересборке или перезапуске контейнеров.
